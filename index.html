Schuyler Lawrence's Portfolio

OmniEats
A Google Map React site that allows users to find and rate nearby restaurants based on food preferences using location tracking. Features include restaurant map cards and tab section, heatmap, and real-time traffic. Built using JavaScript, React, Redux, PostgreSQL, and the Google Maps API.

Journey Wares
A mock e-commerce site with users, products, cart, and checkout. Features include a nav bar with sorting, webpack, Jest tests, and user registration with a PostgreSQL database. Built using JavaScript, React, and Redux.

ACME Schools
A CRUD (Create, Read, Update, Delete) app developed using React, Redux, Node.js, PostgreSQL, Express, and Sequelize. Users can create or delete a student and change the school in which a student is enrolled.

Star Wars Characters
Star Wars Characters allows users to browse through a list of character information from the Star Wars movies. The character information is retrieved from an API. Built using React.
