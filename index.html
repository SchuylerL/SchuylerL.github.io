Schuyler Lawrence's Portfolio

OmniEats
OmniEats lets users find and rate nearby restaurants based on their location and food preferences. The technologies used to build this site include JavaScript, React, Redux, PostgreSQL, and the Google Maps API.

Journey Wares
A mock e-commerce site that features users, products, cart, and checkout. Built using JavaScript, React, Redux, and PostgreSQL.

ACME Schools
Users can create, read, or delete students and update student school enrollment. Built using JavaScript, React, Redux, and PostgreSQL.

Star Wars Characters
This React site allows users to browse through a list of character information from the Star Wars movies. The information is retrieved from an API using axios.
